# 高中阶段计算机科学学习计划

## 角色定义

**教授** — 知名大学计算机科学学科的资深教授，同时也是一位父亲。
**女儿** — 高一学生，对计算机科学、图形学和游戏开发充满热情。好奇心强，喜欢动手实践，善于提出问题。学习风格偏向"做中学"，比起枯燥的理论更喜欢通过项目实践来理解概念。有一定的数学基础，但对抽象概念需要具象化的解释。热爱游戏，梦想是成为一名游戏开发者或图形学研究员。

## 项目目标

为女儿（高一学生，对计算机科学、图形学和游戏开发充满热情）设计一系列系统的课程，帮助她：

1. 了解计算机科学的基础知识
2. 培养对这门学科的持续兴趣
3. 为大学专业学习打下坚实基础
4. 最终在图形学领域深入发展

## 约束条件

- 以欧美知名大学计算机专业课程为参考（MIT、Stanford、CMU、Berkeley 等）
- 主要帮助了解基础、培养兴趣
- 目前不考虑数学类学科（数学基础在学校课程中学习）
- 采用「螺旋式上升」的学习路径：先实践编程，在实践中穿插底层原理

## 课程体系

### 第一学年：编程基础

#### CS101: 编程基础与计算思维
*Programming Fundamentals and Computational Thinking*

| 项目 | 内容 |
|------|------|
| **时间** | 高一上学期 |
| **参考课程** | Harvard CS50、MIT 6.0001 |
| **语言** | Python |

**学习目标：**
- 理解变量、数据类型、运算符等基本概念
- 掌握条件语句、循环结构、函数定义
- 学会使用列表、字典等内置数据结构
- 培养调试程序的能力和耐心

**推荐资源：**
- 《Python编程：从入门到实践》
- Harvard CS50's Introduction to Programming with Python
- MIT 6.0001 (edX)

**实践项目：**
- 猜数字游戏
- 文字冒险游戏
- 期末项目：命令行 RPG 游戏

---

#### CS102: 面向对象编程与软件设计
*Object-Oriented Programming and Software Design*

| 项目 | 内容 |
|------|------|
| **时间** | 高一下学期 |
| **参考课程** | Stanford CS106B |
| **语言** | C++ |

**学习目标：**
- 掌握 C++ 基础语法：变量、指针、引用、数组
- 理解面向对象三大特性：封装、继承、多态
- 学会设计和实现类与对象
- 理解内存管理的基本概念

**推荐资源：**
- 《C++ Primer》
- LearnCpp.com
- The Cherno's C++ Series (YouTube)

**实践项目：**
- 学生成绩管理系统
- 期末项目：2D 控制台游戏（贪吃蛇/俄罗斯方块）

---

### 第二学年：核心计算机科学

#### CS201: 数据结构
*Data Structures*

| 项目 | 内容 |
|------|------|
| **时间** | 高二上学期 |
| **参考课程** | Berkeley CS61B、Stanford CS106B |
| **语言** | C++ |

**学习目标：**
- 深入理解数组和链表的实现原理
- 掌握栈、队列的实现和应用
- 理解树结构：二叉树、二叉搜索树、堆
- 学习图的表示和基本遍历
- 理解哈希表的原理

**推荐资源：**
- 《大话数据结构》
- 浙江大学《数据结构》MOOC
- Visualgo.net（可视化学习）

**实践项目：**
- 实现动态数组（Vector）
- 实现二叉搜索树
- 期末项目：文件系统模拟器

---

#### CS202: 算法设计与分析
*Algorithm Design and Analysis*

| 项目 | 内容 |
|------|------|
| **时间** | 高二下学期（前半） |
| **参考课程** | MIT 6.006、Stanford CS161 |
| **语言** | C++/Python |

**学习目标：**
- 理解时间复杂度和空间复杂度
- 掌握常见排序算法
- 理解递归和分治思想
- 学习 BFS、DFS 搜索算法
- 掌握 A* 寻路算法（游戏开发核心）

**推荐资源：**
- 《算法图解》
- Abdul Bari's Algorithms (YouTube)
- LeetCode（简单难度）

**实践项目：**
- 迷宫求解器（BFS/DFS）
- 井字棋 AI（Minimax）
- A* 寻路算法可视化

---

#### CS203: 计算机系统导论
*Introduction to Computer Systems*

| 项目 | 内容 |
|------|------|
| **时间** | 高二下学期（后半） |
| **参考课程** | CMU 15-213、Berkeley CS61C |
| **语言** | C/C++ |

**学习目标：**
- 理解二进制和数据表示
- 了解 CPU、内存、存储的基本工作原理
- 理解进程和线程的概念
- 了解内存管理的基本原理
- 理解程序的编译和链接过程

**推荐资源：**
- 《编码：隐匿在计算机软硬件背后的语言》
- Crash Course Computer Science (YouTube)
- 《深入理解计算机系统》（选读）

**实践项目：**
- 二进制计算器
- 简单内存分配器模拟
- 多线程下载器

---

### 第三学年：图形学与游戏开发

#### CS301: 计算机图形学基础
*Fundamentals of Computer Graphics*

| 项目 | 内容 |
|------|------|
| **时间** | 高三上学期 |
| **参考课程** | UCSB GAMES101、Stanford CS148 |
| **语言** | C++/GLSL |

**学习目标：**
- 理解图形渲染管线的基本流程
- 学习基本的光照模型
- 理解纹理映射原理
- 能够使用 OpenGL 进行基础图形编程
- 实现简单的 3D 渲染器

**推荐资源：**
- GAMES101: 现代计算机图形学入门（闫令琪）
- Learn OpenGL（中文版）

**实践项目：**
- OpenGL 画三角形
- 简单 3D 场景（立方体旋转、基础光照）
- 软件光栅化渲染器
- 期末项目：简单 3D 场景编辑器

## 教学原则

1. **螺旋式上升** — 先实践，遇到问题时再深入原理
2. **兴趣驱动** — 保护学习热情比完成任务更重要
3. **项目导向** — 每门课程都有可展示的实践项目
4. **循序渐进** — 难度适合高中生，为大学学习铺路

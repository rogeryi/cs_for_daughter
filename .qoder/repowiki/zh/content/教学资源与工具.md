## 课程概述

CS101是一门面向高一学生的编程基础课程，专注于Python编程语言的教学。该课程采用理论与实践相结合的方式，通过15周的学期课程和6周的假期项目，帮助学生建立扎实的编程基础。

### 课程基本信息

| 项目 | 内容 |
|------|------|
| **课程代码** | CS101 |
| **授课对象** | 高一学生 |
| **编程语言** | Python |
| **参考课程** | Harvard CS50、MIT 6.0001 |
| **总课时** | 32.5小时 |

### 课程目标

完成本课程后，学生将能够：
1. 理解计算思维的基本概念
2. 掌握 Python 编程语言的基础语法
3. 运用变量、条件、循环、函数等基本结构编写程序
4. 使用列表和字典组织数据
5. 理解面向对象编程的基本概念
6. 独立完成一个完整的命令行游戏项目

### 课程结构

课程分为两个主要阶段：

**第一阶段：学期课程（15周）**
- 模块一：入门基础（第1-3周）
- 模块二：核心语法（第4-7周）
- 模块三：函数与模块（第8-10周）
- 模块四：面向对象（第11-13周）
- 模块五：综合应用（第14-15周）

**第二阶段：假期项目（6周）**
- 项目名称：《勇者传说》命令行RPG游戏
- 项目目标：综合运用所学知识完成完整的游戏开发

## 开发环境配置指南

### Python 3.10+ 安装

**系统要求**
- 支持的操作系统：Windows、macOS、Linux
- 推荐内存：至少4GB RAM
- 硬盘空间：至少2GB可用空间

**安装步骤**
1. 访问Python官方网站下载最新版本
2. 运行安装程序并确保勾选"Add to PATH"选项
3. 验证安装：在终端中输入 `python --version`
4. 创建虚拟环境：`python -m venv cs101_env`

### VS Code 设置

**推荐插件**
- Python（由Microsoft提供）
- Pylance（智能代码补全）
- Black Formatter（代码格式化）
- Flake8（代码检查）

**VS Code 配置**
1. 打开VS Code，安装上述插件
2. 设置Python解释器为安装的Python 3.10+
3. 配置工作区设置以启用自动格式化
4. 设置代码片段和快捷键

### 开发环境验证

**测试步骤**
1. 创建新文件 `hello_world.py`
2. 输入简单的打印语句
3. 在终端中运行 `python hello_world.py`
4. 检查是否正确输出结果

## 故障排除指南

### 常见安装问题

**Python安装失败**
- **问题症状**：安装程序无法启动或安装失败
- **解决方案**：
  - 以管理员身份运行安装程序
  - 关闭杀毒软件后再安装
  - 检查系统兼容性
  - 清理之前的Python安装

**PATH环境变量问题**
- **问题症状**：命令行无法识别python命令
- **解决方案**：
  - 重新安装Python并勾选"Add to PATH"
  - 手动添加Python路径到系统环境变量
  - 重启命令行窗口
  - 使用完整路径运行Python

### VS Code配置问题

**插件安装失败**
- **问题症状**：无法从扩展市场下载插件
- **解决方案**：
  - 检查网络连接
  - 更换插件源
  - 手动下载插件包
  - 使用离线安装方式

**Python解释器未找到**
- **问题症状**：VS Code无法识别Python环境
- **解决方案**：
  - 在VS Code中手动选择Python解释器
  - 检查Python安装路径
  - 创建并激活虚拟环境
  - 重启VS Code

### 编程错误处理

**语法错误**
- **问题症状**：代码无法运行并显示语法错误
- **解决方案**：
  - 仔细阅读错误信息
  - 检查括号匹配
  - 验证缩进格式
  - 查阅相关语法文档

**逻辑错误**
- **问题症状**：程序可以运行但结果不正确
- **解决方案**：
  - 使用print语句调试
  - 使用Python Tutor可视化
  - 分步执行代码
  - 编写单元测试

**性能问题**
- **问题症状**：程序运行缓慢或内存占用过高
- **解决方案**：
  - 分析算法复杂度
  - 优化数据结构选择
  - 减少不必要的计算
  - 使用缓存机制

## 课程结构与项目设计

### 学期课程详细安排

**模块一：入门基础（第1-3周）**
- **第1周**：编程概念介绍、Python环境搭建、变量基础
- **第2周**：数据类型详解、类型转换、字符串操作
- **第3周**：条件判断、逻辑运算、程序分支

**模块二：核心语法（第4-7周）**
- **第4周**：循环结构、while循环、循环控制
- **第5周**：列表操作、列表遍历、数据管理
- **第6周**：字典应用、数据存储、信息检索
- **第7周**：综合练习、复杂数据结构、项目实战

**模块三：函数与模块（第8-10周）**
- **第8周**：函数定义、参数传递、返回值
- **第9周**：函数设计、作用域、模块化编程
- **第10周**：模块导入、文件操作、数据持久化

**模块四：面向对象（第11-13周）**
- **第11周**：类和对象、构造函数、实例属性
- **第12周**：方法设计、对象交互、封装概念
- **第13周**：继承关系、多态实现、设计模式

**模块五：综合应用（第14-15周）**
- **第14周**：游戏架构设计、软件工程原理
- **第15周**：项目启动、核心框架搭建、需求分析

### 假期项目设计

**项目概述**
- **项目名称**：《勇者传说》命令行RPG游戏
- **项目目标**：综合运用Python知识开发完整游戏
- **项目周期**：6周，每周约2-3小时

**核心代码结构**
```
CS101/
├── project/
│   ├── main.py          # 游戏入口点
│   ├── game.py          # 游戏主控制器
│   ├── character.py     # 角色类定义
│   ├── battle.py        # 战斗系统实现
│   ├── items.py         # 道具系统
│   ├── shop.py          # 商店系统
│   ├── save.json        # 存档数据文件
│   └── README.md        # 项目说明文档
```

**项目里程碑**
- **第1周末**：完成游戏设计文档
- **第2周末**：实现角色系统
- **第3周末**：完成战斗系统
- **第4周末**：实现道具和商店
- **第5周末**：完成游戏主循环
- **第6周末**：项目最终交付

### 学习方法指导

**主动学习原则**
1. **引导而非代劳** — 遇到问题先让学生尝试解决
2. **鼓励试错** — 允许走弯路，错误是最好的老师
3. **及时反馈** — 每周检查，及时纠正不良习惯
4. **保持热情** — 学习兴趣比完成进度更重要

**学习策略**
- **分阶段学习**：按照课程大纲逐步推进
- **理论结合实践**：每学完一个概念立即练习
- **项目驱动学习**：通过实际项目应用所学知识
- **持续反思改进**：定期总结学习效果和方法

**时间管理**
- **固定学习时间**：每天安排固定的编程学习时间
- **合理分配精力**：根据难度调整学习强度
- **定期复习巩固**：每周回顾本周学习内容
- **预习新内容**：提前了解下周学习主题

**学习资源利用**
- **多渠道学习**：结合教材、视频、实践平台
- **互相学习**：与同学讨论交流，互相帮助
- **寻求指导**：遇到困难及时向老师或同学求助
- **拓展视野**：关注编程领域的最新发展

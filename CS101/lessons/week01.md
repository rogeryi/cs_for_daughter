## 教学内容

### 第一部分：计算机如何"思考"（15 分钟）

#### 1.1 什么是编程？

编程就是**用计算机能理解的语言，告诉计算机要做什么**。

想象一下，你要教一个完全不懂中文的外国朋友做菜。你需要：
- 用他能理解的语言
- 给出非常精确的步骤
- 按正确的顺序说明

这就是编程！

#### 1.2 计算机的"大脑"

```
┌─────────────────────────────────────────┐
│                计算机                    │
│  ┌─────────┐    ┌─────────┐    ┌─────┐ │
│  │  输入   │ →  │  处理   │ →  │ 输出│ │
│  │(键盘等) │    │ (CPU)   │    │(屏幕)│ │
│  └─────────┘    └─────────┘    └─────┘ │
└─────────────────────────────────────────┘
```

计算机只能理解 **0 和 1**（开关），但程序员不需要直接写 0 和 1，我们用**编程语言**来写代码，然后由特殊的程序（解释器/编译器）翻译给计算机。

#### 1.3 为什么选择 Python？

| 特点 | 说明 |
|------|------|
| 语法简洁 | 接近自然语言，容易阅读 |
| 功能强大 | 可以做网站、游戏、AI、数据分析 |
| 社区活跃 | 遇到问题容易找到答案 |
| 就业广泛 | 很多公司在使用 Python |

#### 1.4 课堂讨论

- 你觉得生活中有哪些东西是由程序控制的？
- 你玩过的游戏里，哪些功能需要程序员来实现？

### 第二部分：Python 环境搭建与第一个程序（30 分钟）

#### 2.1 安装 Python

**Windows 用户：**
1. 访问 https://www.python.org/downloads/
2. 下载最新版本（推荐 3.11 或更高）
3. 运行安装程序
4. **重要**：勾选 "Add Python to PATH"
5. 点击 "Install Now"

**Mac 用户：**
1. 打开终端
2. 输入 `python3 --version` 检查是否已安装
3. 如未安装，访问 https://www.python.org/downloads/ 下载

#### 2.2 安装 VS Code

1. 访问 https://code.visualstudio.com/
2. 下载并安装
3. 安装 Python 扩展：
   - 打开 VS Code
   - 点击左侧扩展图标（或按 Ctrl+Shift+X）
   - 搜索 "Python"
   - 安装 Microsoft 官方的 Python 扩展

#### 2.3 第一个程序：Hello World

创建一个新文件 `hello.py`，输入以下代码：

```python
print("Hello, World!")
print("你好，世界！")
print("我开始学编程啦！")
```

**运行方式：**
- 在 VS Code 中按 F5 或点击运行按钮
- 或在终端中输入 `python hello.py`

#### 2.4 理解 print() 函数

`print()` 是 Python 的**内置函数**，用于在屏幕上显示信息。

```python
# 输出文字（需要用引号包围）
print("这是一段文字")

# 输出数字（不需要引号）
print(123)
print(3.14)

# 输出多个内容（用逗号分隔）
print("我的年龄是", 16, "岁")
```

#### 2.5 注释

注释是给人看的说明，计算机会忽略它们。

```python
# 这是单行注释，以 # 开头

"""
这是多行注释
可以写很多行
用于解释复杂的代码
"""

print("Hello")  # 这是行尾注释
```

### 第三部分：变量——给数据起名字（30 分钟）

#### 3.1 什么是变量？

变量就像一个**贴了标签的盒子**，用来存放数据。

```
  ┌─────────┐
  │   16    │  ←── 盒子里的数据（值）
  └─────────┘
      ↑
     age      ←── 盒子的标签（变量名）
```

```python
# 创建一个变量
age = 16

# age 是变量名
# = 是赋值符号（不是数学里的"等于"）
# 16 是变量的值
```

#### 3.2 变量的命名规则

| 规则 | 示例 |
|------|------|
| 只能包含字母、数字、下划线 | `name`, `age2`, `my_score` |
| 不能以数字开头 | ❌ `2name` ✅ `name2` |
| 不能使用 Python 关键字 | ❌ `print`, `if`, `for` |
| 区分大小写 | `Name` 和 `name` 是不同的变量 |

**命名建议：**
- 使用有意义的名字：`student_name` 比 `n` 更好
- 使用小写字母和下划线：`my_age`（Python 风格）

#### 3.3 变量的基本操作

```python
# 创建变量
name = "小明"
age = 16
height = 1.75

# 使用变量
print(name)
print("姓名:", name)
print("年龄:", age)

# 修改变量的值
age = 17  # 现在 age 变成了 17
print("明年的年龄:", age)

# 用一个变量创建另一个变量
birth_year = 2024 - age
print("出生年份:", birth_year)
```

#### 3.4 变量的类型

Python 会自动判断变量的类型：

```python
name = "小明"      # 字符串（str）- 文字
age = 16           # 整数（int）- 整数
height = 1.75      # 浮点数（float）- 小数
is_student = True  # 布尔值（bool）- 真/假

# 查看变量类型
print(type(name))    # <class 'str'>
print(type(age))     # <class 'int'>
print(type(height))  # <class 'float'>
```

### 第四部分：课堂练习（15 分钟）

#### 练习：个人信息卡片程序

创建一个程序，显示你的个人信息卡片。

**要求：**
1. 创建至少 5 个变量存储个人信息
2. 使用 `print()` 输出一个格式美观的信息卡片

**示例代码框架：**

```python
# 个人信息卡片
# 作者：[你的名字]
# 日期：[今天的日期]

# 定义变量
name = "你的名字"
age = 你的年龄
school = "你的学校"
grade = "高一"
hobby = "你的爱好"

# 输出信息卡片
print("=" * 30)
print("       个人信息卡片")
print("=" * 30)
print("姓名:", name)
print("年龄:", age)
print("学校:", school)
print("年级:", grade)
print("爱好:", hobby)
print("=" * 30)
```

**扩展挑战：**
- 添加更多信息（身高、喜欢的游戏、未来梦想等）
- 尝试让卡片更美观（使用不同的符号做边框）

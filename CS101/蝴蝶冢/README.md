# 蝴蝶冢 (Butterfly's Tomb)

## 游戏简介

一款恐怖解密类文字 RPG 游戏。在架空世界的雪原森林深处，罗罗诺亚·索隆与霜月古伊娜被困在一个被蝴蝶缠绕的小镇，探寻死亡与记忆的秘密。

## 核心设定

- **蝴蝶现象**: 人死后，尸体会飞出蝴蝶，每只蝴蝶都携带着逝者的记忆碎片
- **故事背景**: 雪原森林深处的小镇"蝶栖镇"，与世隔绝，充满诡异的平静
- **文本风格**: 《百年孤独》式的魔幻现实主义叙事
- **游戏风格**: 《锈湖》式的超现实解密体验

## 主要角色

| 角色 | 身份 |
|------|------|
| 罗罗诺亚·索隆 | 主角，剑士，陷入谜局 |
| 霜月古伊娜 | 索隆的同伴，神秘失踪 |
